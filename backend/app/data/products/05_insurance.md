# 전북은행 · 광주은행 보험 (방카슈랑스) 상품

---

### JB 실버 종합건강보험 (방카슈랑스)
카테고리: 보험
은행: 전북은행
대상: 만 50~70세 개인
상품유형: 실손 + 진단비 통합 건강보험 (방카슈랑스)
보험료: 월 3~8만 원 (나이·성별·가입금액에 따라 다름)
보장내용
- 실손 의료비: 입원·통원 의료비 실비 보장 (자기부담금 20%)
- 암·심근경색·뇌졸중 3대 중증질환 진단비 최대 2,000만 원
- 골절·화상 상해 보장 포함
특징
- 고령자도 가입 가능 (비흡연자 우대)
- 갱신형으로 초기 보험료 부담 낮음
- 전북은행 계좌 자동이체 시 0.5% 보험료 할인

### 광주은행 노후 생활비 연금보험 (방카슈랑스)
카테고리: 보험
은행: 광주은행
대상: 만 40~60세 개인
상품유형: 즉시연금 / 거치형 연금보험
납입기간: 일시납 또는 5·10·20년납
기대 연금: 보험료·나이·납입기간에 따라 산출
보장내용
- 사망 시 납입 보험료 환급 (또는 기납입 보험료 이상 보장)
- 종신형 / 확정기간형 중 선택
특징
- 공시이율 적용 (현재 연 3.0% 수준, 최저보증이율 1.5%)
- 10년 이상 유지 시 이자소득세 비과세 (세법상 요건 충족 시)
- 노후 월 생활비 보완 목적에 적합

### JB 간병인 지원 장기요양보험 (방카슈랑스)
카테고리: 보험
은행: 전북은행
대상: 만 40~65세 개인
상품유형: 장기요양보험
보험료: 월 1~4만 원 (나이·성별 기준)
보장내용
- 장기요양 1~5등급 판정 시 매월 간병 생활비 지급 (최대 월 100만 원)
- 치매·중풍 진단 시 추가 일시금 지급 (최대 500만 원)
- 중증 요양 등급 시 간병인 파견 서비스 연계
특징
- 고령화 사회 핵심 리스크인 장기요양 비용 대비
- 단독 가입 또는 실손보험과 함께 가입 시 보완 효과
- 보험료 납입 기간 중 세액공제 혜택 (연 100만 원 한도)
